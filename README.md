# 匿名チャットサイト

好きな名前・ID・アイコンを設定して、リアルタイムチャットと通話ができる匿名チャットサイトです。

## 機能

- ✨ 自由な名前・ID・アイコン設定
- 🚪 4つの部屋システム
- 💬 リアルタイムチャット
- 📞 WebRTC通話機能（マイク/スピーカーミュート、個別音量調整）
- 🚫 ブロック機能
- 🎨 カスタマイズ可能な文字色・背景色
- 🔍 メッセージ・ユーザー検索
- 👥 オンラインユーザー一覧表示

## セットアップ

### 1. 依存関係のインストール

```bash
# ルートディレクトリ
npm install

# サーバー
cd server
npm install

# クライアント
cd ../client
npm install
```

### 2. 開発サーバーの起動

ルートディレクトリで実行：

```bash
npm run dev
```

これで以下が起動します：
- バックエンドサーバー: http://localhost:3000
- フロントエンド: http://localhost:5173

### 3. ブラウザでアクセス

http://localhost:5173 を開いてください。

## Vercelデプロイ

Vercelでデプロイする場合：

1. Vercelのプロジェクト設定で「Root Directory」を`client`に設定
2. または、`vercel.json`の設定を使用（既に設定済み）

## 使い方

1. **ユーザー登録**
   - 表示名、ID、アイコン、色を設定
   - 「チャットを始める」をクリック

2. **部屋選択**
   - 4つの部屋から選択
   - 部屋ごとに独立したチャット

3. **チャット**
   - メッセージを入力して送信
   - 他のユーザーとリアルタイムでチャット
   - 検索バーでメッセージ・ユーザーを検索

4. **通話**
   - 右側のユーザーリストから通話したい相手を選択
   - 📞ボタンをクリックして通話開始
   - マイク/スピーカーミュート、音量調整が可能

5. **ブロック**
   - ユーザーリストやメッセージから🚫ボタンでブロック
   - メニューからブロック管理

6. **設定**
   - 右上のメニューボタンから設定
   - 文字色・背景色のカスタマイズ

## 技術スタック

- **フロントエンド**: React + TypeScript + Vite + Tailwind CSS
- **バックエンド**: Node.js + Express + Socket.io
- **通話**: WebRTC

## 注意事項

- 通話機能はHTTPS環境またはlocalhostでのみ動作します
- マイクへのアクセス許可が必要です
- 複数のブラウザタブでテストできます
